# Getting Started

To run this application, ensure the following prerequisites are met:

## Prerequisites

- A **Minecraft 1.8.9 Vanilla Server** must be running in the background. (server only, not a single-player world opened to LAN)
- The server must listen on port `25565` (default).

## Configuration

If you need to use a different port, you can change the port setting in `Game.cs`.

> **Note:** After modifying the port in `Game.cs`, make sure your Minecraft server also uses the same port.
